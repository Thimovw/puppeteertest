Security Policy

We nemen beveiliging serieus en implementeren industry-standard security practices.

Beveiligingsmaatregelen

  • SSL/TLS encryptie voor alle verbindingen
  • Two-factor authenticatie (2FA)
  • Regular security audits en penetration testing
  • Data encryption at rest en in transit
  • ISO 27001 gecertificeerd

Incident Response

Bij een security incident nemen we onmiddellijk actie en informeren we betrokken partijen binnen 72 uur.
